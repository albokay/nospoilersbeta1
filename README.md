# No-Spoilers Full UI (V07.2)

This version includes your full single-file UI so you see the forum immediately at http://localhost:5173/.

## Run
1) Install Node 18+
2) In this folder:
   ```bash
   npm install
   npm run dev
   ```
3) Open the printed localhost URL.

> Styling note: This build uses simple inline CSS instead of Tailwind. Functionality is the same.
